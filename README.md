

# picsarthomework

Repository with JavaScript homework and practice projects — covering core JS fundamentals, and data structures & algorithms.

## 📁 Structure

```
picsarthomework/
├── JSMonth1/      # Month 1 JavaScript coursework and exercises
├── coreJS/        # Core JavaScript concepts and practice tasks
└── DSA/           # Data Structures & Algorithms
    ├── DS/         # Data structure implementations (linked list, deque, stack, queue, etc.)
    └── Algorithms/ # Algorithm implementations
```

## 📚 Topics Covered

- **JavaScript Fundamentals** — variables, functions, scope, closures, prototypes
- **Core JS** — async/await, promises, the event loop, ES6+ features
- **Data Structures** — singly/doubly linked lists, circular linked lists, stacks, queues, deques (including bucketed deque implementation), etc.
- **Algorithms** — sorting (merge sort, etc.), searching, two-pointer techniques, recursion
- **LeetCode Practice** — solved problems involving linked lists, arrays, and more

## 🛠️ Tech Stack

- JavaScript (Node.js)

## 🚀 Getting Started

Clone the repository:

```bash
git clone <repo-url>
cd picsarthomework
```

Run any file with Node.js:

```bash
node DSA/Algorithms/mergeSort/taskA.js
```
